# 🧠 Lógica de Programação - Portugol (VisualG)

Este repositório contém exercícios e algoritmos desenvolvidos em **Portugol utilizando o VisualG** durante meus estudos de **Lógica de Programação**.

O objetivo deste repositório é documentar minha jornada de aprendizado, praticar conceitos fundamentais de programação e construir uma base sólida para o desenvolvimento em outras linguagens no futuro.

## 📚 Sobre

Aqui você encontrará diversos algoritmos e exercícios focados em conceitos essenciais de lógica de programação, como:

- Variáveis e tipos de dados
- Estruturas condicionais (se / senão)
- Estruturas de repetição (para / enquanto / repita)
- Operações matemáticas
- Entrada e saída de dados
- Raciocínio lógico e resolução de problemas

Todos os códigos foram desenvolvidos utilizando **Portugol** no ambiente **VisualG**, muito utilizado no ensino de lógica de programação para iniciantes.

## 🎯 Objetivo

Os principais objetivos deste repositório são:

- Praticar lógica de programação
- Organizar os exercícios do curso
- Acompanhar minha evolução como desenvolvedor
- Contribuir para meu portfólio de programação

## 🛠️ Tecnologias

- **Portugol**
- **VisualG**
